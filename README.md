
# MavenDeploy

支持上传多种maven仓库,配置相关全部自动生成，无需在项目中增加上传配置，支持使用maven、artifactory插件swing界面上传

## 安装指南

通过jar包在项目根目录直接运行使用

```
java -jar deploytools.jar
```

安装插件使用，也可以通过搜索MavenDeploy进行安装
```
Settings>Plugins>install plugin from disk
```
